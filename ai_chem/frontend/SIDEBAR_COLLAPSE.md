# 轨道可视化侧边栏折叠功能

## 访问地址

**http://localhost:5175/chem/**

然后导航到轨道可视化页面（OrbitalViewer）

## 新增功能

### 可折叠侧边栏

左侧控制面板现在支持折叠/展开功能：

#### 折叠按钮
- 位置：左侧边栏右上角
- 样式：渐变色按钮（青色到紫色）
- 图标：`←` 折叠 / `→` 展开
- 悬停效果：放大 + 发光效果

#### 折叠状态
- **展开状态**（默认）：
  - 侧边栏宽度：300px
  - 显示所有控制选项
  - 完整的标题和说明

- **折叠状态**：
  - 侧边栏宽度：50px
  - 只显示折叠按钮
  - 内容隐藏，节省空间
  - 主可视化区域自动扩展

#### 动画效果
- 平滑过渡动画：0.3秒
- 宽度、内边距同步变化
- 内容淡入淡出效果

## 现有功能保持不变

### 轨道选择
- 支持 1s 到 7p 的所有轨道类型
- 包括 s、p、d、f 轨道
- 每个轨道有独特的颜色方案

### 参数调节
- 粒子数量：10,000 - 100,000
- 粒子大小：0.01 - 0.2
- 粒子透明度：0.1 - 1.0
- 发光强度：0.5 - 3.0

### 轨道信息面板
- 位置：右上角（独立于左侧边栏）
- 也支持折叠/展开
- 显示量子数、节点数等信息

### 3D 交互
- 鼠标拖动旋转
- 滚轮缩放
- 自动阻尼效果

## 技术实现

- 使用 Vue 3 响应式状态管理
- CSS transition 实现平滑动画
- 条件渲染优化性能
- Three.js 渲染不受影响

## 使用建议

1. 首次进入时侧边栏展开，方便选择轨道
2. 选择好参数后可以折叠侧边栏，获得更大的可视化空间
3. 需要调整参数时再展开侧边栏
4. 折叠按钮始终可见，随时切换
