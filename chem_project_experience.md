# ChemTutor - 智能化学教育平台

> 项目时间：2026年1月 - 至今 | 项目角色：全栈开发负责人

---

## 项目概述

ChemTutor 是一个面向高中化学教育的综合性智能学习平台，集成了 **AI 智能问答**、**3D 分子可视化**、**图像识别** 等多种功能，旨在提供直观、交互式的化学学习体验。项目采用微服务架构，支持本地部署和 Docker 容器化部署。

### 三大核心模块

| 模块 | 功能描述 | 关键技术 |
|------|----------|----------|
| **AI 智能问答** | RAG 检索增强生成，基于教材内容回答问题 | Jieba 分词、关键词匹配、TF-IDF |
| **3D 分子可视化** | 支持多种输入格式的分子/晶体结构 3D 渲染 | RDKit、ETKDG、XTB 量子化学、3Dmol.js |
| **图像识别** | 上传化学结构图片自动识别为 SMILES | DECIMER 深度学习模型 |

## 核心功能实现

### 1. AI 智能问答系统
- 基于 RAG（检索增强生成）架构，实现智能问答
- 使用 Jieba 分词提取关键词，TF-IDF 计算相关性
- 支持教材内容索引、来源追溯、对话历史记录
- 前后端分离架构，Vue.js + FastAPI 实现 RESTful API

### 2. 3D 分子可视化
- 支持多种输入格式：SMILES、中英文名称、化学式
- 有机分子：使用 ETKDGv3 算法生成初始构型，MMFF 力场优化
- 无机分子：集成 XTB 半经验量子化学方法进行高精度几何优化
- WebGL 实时渲染，支持旋转、缩放、平移等交互操作

### 3. 图像识别
- 基于 DECIMER 深度学习模型识别化学结构式
- 自动将图片转换为标准 SMILES 字符串
- 集成 RDKit 进行分子结构验证和可视化

## 开发难点与解决方案

| 难点 | 解决方案 |
|------|----------|
| **无机分子 3D 结构生成** | 引入 XTB + ASE 半经验量子化学方法，替代传统力场，提升结构精度 |
| **RAG 检索相关性不足** | 设计多维度评分算法（完全匹配+部分匹配+同句关键词聚合），显著提升检索质量 |
| **前端构建内存溢出** | 调整部署策略，本地构建前端静态文件，服务器仅托管，避免 OOM |
| **Docker 依赖冲突** | 为 RDKit 定制 Dockerfile，添加 libxrender1、libxext6 等系统库 |

## 开发流程

1. **需求分析**：确定三大核心模块功能边界与技术选型
2. **架构设计**：采用微服务架构，各模块独立部署，通过 Nginx 统一代理
3. **后端开发**：FastAPI/Flask 实现 RESTful API，RDKit 处理化学计算
4. **前端开发**：Vue.js 3 + TypeScript，Vite 构建优化
5. **容器化部署**：编写 Dockerfile 和 docker-compose.yml，实现一键部署
6. **测试与优化**：单元测试、接口测试、性能优化
